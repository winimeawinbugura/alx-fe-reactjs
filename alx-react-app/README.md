## Project Overview

This project sets up a **ReactJS app** with Vite as the build tool. Vite provides a **lightning-fast development environment** with hot module replacement (HMR) and optimized builds. By completing this project, you will have a **working ReactJS frontend environment** ready for development.

---

## Objectives

- Learn how to create a **GitHub repository** for a frontend project.
- Initialize a **React app** using **Vite**.
- Install necessary **dependencies** and run the development server.
- Verify that the React app runs successfully in the browser.
- Understand the **project directory structure** and purpose of key files.
- Prepare for further **ALX ReactJS projects**.

- [Node.js](https://nodejs.org) (LTS version recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- A modern web browser (Chrome, Edge, Firefox, or Safari)
- Git (for cloning and pushing repositories)
